LinkedIn AI Post Engine

Personal Automation Project | ⚡ Deploying Soon

📋 Google Sheets  →  🤖 GPT-4 (polish + format)  →  🖼️ DALL·E 3 (visual)  →  in LinkedIn API

The Real Problem

Professionals learn every day but never share it — because writing a polished LinkedIn post takes 30–45 minutes of editing, formatting, and generating visuals. The friction kills consistency, and inconsistency kills audience growth.

How It Works

You type raw bullet-point notes into a Google Sheet — no grammar concerns, no formatting. n8n watches the sheet, feeds your notes to GPT-4 to generate an engaging, on-brand LinkedIn post, triggers DALL·E 3 to create a matching visual, and publishes everything to LinkedIn automatically.

~3 hrs
Saved per week
0 min
Manual editing time

Key Features

  • Google Sheets trigger — just drop in bullet points, no app switching.
  • GPT-4 rewrites notes into professional, engaging LinkedIn-optimized posts.
  • DALL·E 3 generates a contextual image automatically for each post.
  • Publishes directly via LinkedIn API — zero manual steps.
  • Fully customizable tone, format templates, and posting schedule.

Technologies Used

n8n GPT-4 API DALL·E 3 Google Sheets API LinkedIn API

Freelance Client Radar

Personal Automation Project | ⚡ Deploying Soon

🌐 Upwork · Fiverr · LinkedIn · Reddit  →  🔍 Scrape  →  🤖 AI Score  →  📊 Daily Digest Sheet

The Real Problem

Freelancers waste 5+ hours per week manually browsing Upwork, Fiverr, Reddit, and LinkedIn hunting for relevant projects. Most leads are stale or off-niche by the time they find them — a huge opportunity cost that kills earning potential.

How It Works

The workflow runs on a schedule, simultaneously scraping Upwork, Fiverr, LinkedIn Jobs, and targeted Reddit threads. Each listing is fed to an AI scoring model that rates relevance, budget fit, and competition level. Only the best leads are surfaced in a structured daily digest — ready to act on.

Saved per week
4+

Key Features

  • Parallel scraping across Upwork, Fiverr, LinkedIn, and niche Reddit communities.
  • AI relevance scoring filters out off-niche, low-budget, or expired listings.
  • Structured daily digest delivered to Google Sheets with budget, niche, and platform data.
  • Keyword and skill-set filters — fully customizable per freelancer profile.
  • Runs automatically on a schedule — zero manual platform browsing required.

Technologies Used

n8n Web Scraping GPT-4 API Google Sheets API Upwork API LinkedIn API
